DHTML

Dynamic HTML (DHTML) refers to a Web client's ability to update its local document object model (DOM) after an HTTP request is completed. Rather than requiring a browser to make a new HTTP request whenever it needs to update its rendered content, DHTML can be used to significantly improve page response time.
