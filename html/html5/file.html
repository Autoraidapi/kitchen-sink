This page demonstrates how to use the HTML5 FileReader object to retrieve file information about selected files. Note: If you select files from the shared folder on BlackBerry Tablet OS, you must ensure that the blackberry-tablet.xml file (found within bbwp/AirAppTemplates/src) includes the following permission:

<permission>access_shared</permission>
